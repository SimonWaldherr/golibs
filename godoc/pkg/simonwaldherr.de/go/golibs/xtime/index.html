func Fmt ¶ 

func Fmt(format string, t time.Time) string

Fmt (xtime.Fmt) is an alias for StrfTime

▹ Example

▾ Example

Code:

t := time.Unix(1234567890, 0)
cet, _ := time.LoadLocation("CET")
t = t.In(cet)
fmt.Println(xtime.Fmt("%Y-%m-%d %H:%M:%S", t))

Output:

2009-02-14 00:31:30
